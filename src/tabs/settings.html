Welcome to Task Manager

v0.1.3

Presented to you by Team Forkbomb

This app helps people with ADHD (or those prone to procrastination) get more done, thus alleviating anxiety and pressure. The program’s killer features include the “Help!” tab that suggests tasks when you’re overwhelmed or having trouble getting started, a notification system that checks in on you and keeps you focused, and the task/project planner for breaking down bigger projects into smaller tasks.

Current Features

  • Task Tab - Contains a list view of all tasks and allows you to add tasks to your task lists
  • Planner Tab - Contains a weekly view of tasks that allows you to look forward and backward
  • Task Planner Tab - Allows you to plan and create subtasks under existing tasks
  • Time Block Tab - Contains an alternating worktime/breaktime timer
  • Help Tab - Recommends you certain tasks from your task list based on certain scenarios
  • Reminders Tab - Contains previous task reminders
  • Settings Tab - Allows you to customize the app such as
    • Notification frequency and timing
    • Tabs Displayed
